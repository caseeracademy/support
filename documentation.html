Caseer Academy API

API Introduction

This documentation provides a detailed overview of the custom REST API for the Caseer Academy mobile application. The API is designed to provide a clean and efficient interface for the app to interact with the WordPress backend, which uses Tutor LMS and WooCommerce.

Base URL: https://caseer.academy/wp-json/my-app/v1

Authentication

Most endpoints require authentication via a JSON Web Token (JWT). To get a token, send a user's credentials to the JWT plugin endpoint. This token must then be included in the `Authorization` header for all protected requests as a Bearer token.

POST/token

Authenticates a user and returns a JWT.

FULL URL

https://caseer.academy/wp-json/jwt-auth/v1/token

REQUEST BODY (form-data)

username string, required

password string, required
